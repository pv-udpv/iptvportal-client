## How to Contribute

We welcome contributions from the community! Here's how you can help:

### Ways to Contribute

1. **Report Bugs** - Found a bug? [Open an issue](https://github.com/pv-udpv/iptvportal-client/issues/new)
2. **Suggest Features** - Have an idea? [Start a discussion](https://github.com/pv-udpv/iptvportal-client/discussions)
3. **Submit Pull Requests** - See our [Contributing Guide](.github/CONTRIBUTING.md)
4. **Improve Documentation** - Help make our docs better
5. **Write Tests** - Increase code coverage
6. **Review Code** - Help review pull requests

### Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/iptvportal-client.git`
3. Create a feature branch: `git checkout -b feature/amazing-feature`
4. Make your changes
5. Run tests: `make test`
6. Run linters: `make lint`
7. Commit your changes: `git commit -m 'Add amazing feature'`
8. Push to your fork: `git push origin feature/amazing-feature`
9. Open a Pull Request

### Code of Conduct

Please be respectful and constructive in all interactions. We aim to foster an open and welcoming environment.

### Recognition

All contributors will be acknowledged here and in release notes. Thank you for your contributions!
